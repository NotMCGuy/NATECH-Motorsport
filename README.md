# NATECH Motorsport

This repository now contains only the native Dash OS runtime under `dash_os/`.

## Primary runtime
- `dash_os/` -> Embedded dashboard application foundation (non-web)

## Quick start
1. `cd dash_os`
2. `python -m pip install -e .`
3. `python -m natech_dash_os.main --simulate`

For full runtime and ignition/boot-video test instructions, see `dash_os/README.md`.
